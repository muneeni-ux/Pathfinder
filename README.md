 What the Website Should Contain (Key Sections)
1.	 Home Page
o	Vision, mission, and Pathfinder @75 overview.
2.	 About the Project
o	Background, objectives, church connection, and expected outcomes.
3.	 Get Involved / Registration
o	Registration forms for clubs, volunteers, ambassadors, and partners.
4.	 Tree Planting Progress Tracker
o	Display trees planted per club/county/conference with maps or counters.
5.	 Media & Stories
o	Photos, videos, testimonies, downloadable reports.
6.	 Resources & Downloads
o	Guidelines, proposal documents, training materials, sermons, and branding kits.
7.	 Donation & Sponsorship Page
o	Options for mobile money, bank, or partnership pledges.
8.	 Contact & Support
o	Email, phone, coordinators’ details, WhatsApp link, etc.

 Key Functionalities the Website Should Have
1. Online Registration System
•	For clubs, leaders, sponsors, volunteers, and events.
2. Tree Count/Impact Dashboard
•	Display total trees planted.
•	Track by region, conference, or club.
3. Donation & Partnership Module
•	Payment integration (M-Pesa, PayPal, bank).
•	Sponsorship categories.
4. News & Updates Section
•	Blog-style posts for events, reports, and testimonies.
5. Photo/Video Gallery
•	Visual impact and sharing success.
6. Document Center
•	Policy documents, training, proposals, certificates, and reports.
7. Multilingual Options (optional)
•	English, Swahili,  (for East/Central Africa).
8. Social Media Integration
•	Links to Facebook, WhatsApp, YouTube, Instagram, etc.
